# AI ShieldPro Coming Soon Site

This folder contains a standalone static landing page for `aishieldpro.com`.

## Files

- `index.html`: page structure and content
- `styles.css`: animated visual design
- `script.js`: small entrance animation trigger

## Free hosting options

This site is plain static HTML/CSS/JS, so it can be deployed on free static hosting platforms such as Cloudflare Pages, GitHub Pages, Netlify, or Vercel.
